🎡 什么是 Wheel（轮动）策略？ ▼

Wheel 策略是一种通过反复卖出期权来被动赚取权利金的方法，适合长期看好 BTC 的投资者。

  • 第一步（有 USDT 时）：卖出低于当前价格的 Put 期权，收取权利金
  • 被行权时：以行权价买入 BTC（比你预计更便宜），同时已收了权利金
  • 持有 BTC 时：卖出高于当前价格的 Call 期权，继续收权利金
  • 被行权时：以行权价卖出 BTC（比你预计更高），再次获利，回到有 USDT 的状态

核心思路：无论涨跌，每步都能收权利金。当然，如果 BTC 大幅暴跌，持有 BTC 的账面亏损会超过权利金收入，这是最主要的风险。

💛 Sell Put 详解（有 USDT 时用） ▼

卖出看跌期权：你卖给别人一个权利，对方可以在到期前以行权价把 BTC 卖给你。

📌 举例：BTC 现价 $80,000
你卖出一张 行权价 $75,000、14 天后到期的 Put
收到权利金：$400

情况一（BTC 到期 > $75,000）：期权作废 ✅ 你净赚 $400
情况二（BTC 到期 = $73,000）：你被迫用 $75,000 买入 BTC
  但实际成本 = $75,000 - $400 = $74,600
  账面浮亏：$73,000 - $74,600 = -$1,600
  • 最大收益：权利金（BTC 未跌至行权价）
  • 最大亏损：行权价 × 合约数量（BTC 跌至 0，极端情况）
  • 建议：行权价选在当前价格 10-20% 以下，Delta 约 -0.20
💜 Covered Call 详解（持有 BTC 时用） ▼

备兑看涨期权：持有 BTC 的情况下，卖出高于当前价格的 Call 期权，用持仓作担保。

📌 举例：你持有 1 BTC，现价 $80,000
你卖出一张 行权价 $90,000、21 天后到期的 Call
收到权利金：$600

情况一（BTC 到期 < $90,000）：期权作废 ✅ 你净赚 $600，BTC 照持
情况二（BTC 到期 = $95,000）：你被迫以 $90,000 卖出 BTC
  实际卖出价 = $90,000 + $600 = $90,600
  虽然错过了 $95,000，但比没卖 Call 多赚了 $600
  • 最大收益：（行权价 - 买入成本）× 合约数 + 权利金
  • 主要风险：BTC 暴跌时，权利金不足以覆盖持仓亏损
  • 建议：行权价选在当前价格 10-20% 以上，Delta 约 0.20
📊 如何看懂 Delta / IV / IVP 等指标？ ▼
  • Delta（-1 到 0 / 0 到 1）：反映被行权的概率。Put 的 Delta -0.20 意味着约 20% 概率被行权。越接近 0 越安全，收益也越低。
  • IV（隐含波动率）：市场预期 BTC 未来波动幅度。IV 越高，期权越贵，卖方收益越高。
  • DVOL：Deribit 的 BTC 波动率指数，类似股市的 VIX。50 以下偏低，80 以上偏高。
  • IVP（IV 历史百分位）：当前 DVOL 在过去一年处于多少百分位。IVP > 70 说明波动率处于历史高位，是卖出期权的好时机。
  • Theta：每天因时间流逝损失的期权价值。作为卖方，Theta 是你的朋友——每天都在帮你赚钱。
